# Alert Forge

No-code Telegram alert builder. Monitor wallets, tokens, gas prices, and custom APIs.

## Features

- Wallet activity monitoring
- Token price alerts
- Gas price tracking
- Custom API monitoring

## Tech Stack

- Frontend: Next.js + TypeScript + Tailwind
- Backend: Python + Telegram Bot API
- Database: Supabase
- Deployment: Railway + Vercel
